# ✅ قائمة مراجعة البريد والإشعارات للمسؤول

## 📧 الإعدادات الأساسية

- [ ] تم الحصول على مفتاح Resend API من https://resend.com
- [ ] تم إضافة RESEND_API_KEY في .env.local
- [ ] تم تحديث ADMIN_EMAIL ببريد المسؤول الصحيح
- [ ] تم التحقق من NEXT_PUBLIC_SITE_URL (للروابط في البريد)
- [ ] تم تثبيت حزمة resend: `npm install resend`

## 🔧 معالجة المشاكل

### البريد لا يصل؟
```
1. تحقق من RESEND_API_KEY صحيح
2. تحقق من البريد ليس في Spam
3. تفقد Resend logs على https://resend.com/logs
4. تحقق من عدم كون البريد في القائمة السوداء
```

### البيانات لا تحفظ؟
```
1. تحقق من اتصال Supabase
2. تحقق من صحة بيانات الاعتماد
3. اطلع على Supabase logs
4. تحقق من الأذونات في Supabase
```

### النموذج يعطي خطأ؟
```
1. افتح Browser Console (F12)
2. تحقق من أي أخطاء في Network
3. تفقد رسالة الخطأ المعروضة
4. جرب بتحديث الصفحة
```
